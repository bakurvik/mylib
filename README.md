# mylib

## library
Microservice that stores books' and authors' data. API:

### POST /api/authors
Creates new author

### GET /api/authors
Gets all authors with short info

### GET /api/authors/{id}
Gets an author's full info

### DELETE /admin/authors/{id}
Deletes an author

### PUT /api/authors
Update an author's info
